## 📝 紀錄格式

每份會議紀錄包含：

| 欄位 | 說明 |
|------|------|
| **會議主題** | 本次會議主要議題 |
| **日期時間** | 會議時間 (UTC+8) |
| **與會 Agent** | 參與討論的 Agent |
| **用戶指令** | 先生的原始指令 |
| **討論內容** | 各 Agent 發言紀錄 |
| **不同意見** | 若有分歧，記錄各方觀點 |
| **最終決策** | JARVIS 的最終決定 |
| **行動項目** | 後續執行事項 |
