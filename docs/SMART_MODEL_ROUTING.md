# Smart Model Routing - 智能模型路由

**最後更新：** 2026-03-06 13:56

---

## 🎯 目標

根據問題複雜度自動選擇最適合的模型：
- **簡單問題** → qwen3.5-plus (快速)
- **複雜問題** → qwen3-max (深度思考)

---

## 🤖 模型選擇規則

### 使用 qwen3-max (複雜問題)

**觸發條件：**
- 包含複雜關鍵詞：分析、策略、預測、評估、複雜、深度、多步驟...
- 問題長度 > 200 字元
- 包含多個問題
- 複雜度評分 >= 5

**適用場景：**
- 投資組合分析
- 風險評估報告
- 地緣政治分析
- 趨勢預測
- 架構設計
- 代碼重構/優化

### 使用 qwen3.5-plus (簡單問題)

**觸發條件：**
- 包含簡單關鍵詞：查詢、價格、狀態、更新、推送...
- 問題長度 < 100 字元
- 單一問題
- 複雜度評分 < 5

**適用場景：**
- 股價查詢
- 狀態檢查
- 快速報告
- 定時任務
- Git 推送
- Telegram 通知

## 📊 複雜度評分規則

| 因素 | 加分 | 說明 |
|------|------|------|
| 複雜關鍵詞 | +2 每個 | 分析、策略、預測等 |
| 簡單關鍵詞 | -1 每個 | 查詢、價格、狀態等 |
| 長度 > 200 | +3 | 長問題通常需要深度思考 |
| 長度 > 100 | +1 | 中等長度 |
| 每個問號 | +1 | 多問題表示複雜 |

**評分 >= 5** → 使用 qwen3-max  
**評分 < 5** → 使用 qwen3.5-plus
